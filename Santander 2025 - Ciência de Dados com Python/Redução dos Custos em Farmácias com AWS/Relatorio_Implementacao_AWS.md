# RELATÓRIO DE IMPLEMENTAÇÃO DE SERVIÇOS AWS

Data: 23/01/2026

Empresa: Abstergo Industries

## Introdução
Este relatório apresenta o processo de implementação de ferramentas na empresa **Abstergo Industries**, realizado por **Igor Santos**.  
O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de realizar diminuição de custos imediatos e iniciar a jornada de migração para a nuvem, uma vez que a empresa não possuía nenhuma solução cloud anteriormente.

## Descrição do Projeto
O projeto de implementação de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos específicos. A seguir, serão descritas as etapas do projeto:

### Etapa 1:
- **Nome da ferramenta:** Amazon EC2  
- **Foco da ferramenta:** Hospedagem de servidores e aplicações  
- **Descrição de caso de uso:** Utilização do Amazon EC2 para substituir servidores físicos locais, reduzindo custos com aquisição de hardware, manutenção e energia elétrica. A empresa poderá hospedar sistemas internos e aplicações corporativas de forma escalável e sob demanda.

### Etapa 2:
- **Nome da ferramenta:** Amazon S3  
- **Foco da ferramenta:** Armazenamento de dados  
- **Descrição de caso de uso:** Implementação do Amazon S3 para armazenar documentos, relatórios, backups e dados operacionais. O serviço oferece alta durabilidade, baixo custo e elimina a necessidade de servidores de arquivos locais.

### Etapa 3:
- **Nome da ferramenta:** Amazon RDS  
- **Foco da ferramenta:** Banco de dados gerenciado  
- **Descrição de caso de uso:** Migração dos bancos de dados locais para o Amazon RDS, reduzindo custos operacionais com administração, garantindo alta disponibilidade, backups automáticos e maior segurança dos dados.

## Conclusão
A implementação de ferramentas na empresa **Abstergo Industries** tem como esperado a redução de custos com infraestrutura, maior escalabilidade, segurança e confiabilidade dos sistemas, além de preparar a empresa para atuar como revendedora de soluções para outras empresas e farmácias.  
Recomenda-se a continuidade da utilização das ferramentas implementadas e a busca por novas tecnologias que possam melhorar ainda mais os processos da empresa.
